# Domain-records-tester

This is a tool which is used to check the records of a domain!!

### Usage

```
git clone https://github.com/SHXMIM/Domain-records-tester.git
```
```
cd Domain-records-tester
```
```
chmod 755 Domain-records-tester.sh 
```
```
./Domain-records-tester.sh 
``` 
### Example

```
./Domain-records-tester.sh  example.com
```
### Output
```

[*] Finding AAAA Record

[*] Findind Cname Record
```
